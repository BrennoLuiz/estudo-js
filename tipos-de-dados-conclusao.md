# Tipos de Dados

Conforme o ECMAScript standard temos 9 tipos de dados:

* Data types
    * Primitive / Primitive value (Não sofrem alterações)
    * Structural
    * Structural Primitive

## Primitivos

* String
* Number
* Boolean
* Undefined
* Symbol
* BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Esturutral / Structural Root Primitive

* null
